# 🎮 Maikudoku – Simple, Fun, and Free Sudoku

Welcome to **Maikudoku**, the Sudoku app you can play anywhere, without distractions. Whether you're a beginner or an expert, you'll always find a challenge that matches your level.

---

## ✨ Main Features

🎯 **Difficulty levels**  
Choose between Easy, Medium, Hard, Really Hard and Super Hard – a new grid awaits you every time you open the app.

🔄 **Infinite puzzles**  
You'll never run out of games to play: every match generates a unique new puzzle.

🌙 **Modern Design (Material You)**  
The interface adapts to your phone and your preferred theme (light or dark) for a comfortable experience.

📱 **Play Offline**  
No internet needed. Open the app and play whenever and wherever you want.

🚫 **No Ads**  
Play uninterrupted: your favorite game without any ads or distractions

✅ **Real-Time Validation**  
The app tells you immediately if a number is invalid, so you learn as you play.

## 📥 How to Download and Install

### Via GitHub (manual method)

1. Go to the **"Releases"** section of the project on GitHub.
2. Download the `app-debug.apk` file from the latest version.
3. Open the file on your Android phone and follow the installation instructions.

**Note:** On Android 8.0 and later, you may need to authorize the app to install from "Unknown Sources" in your phone's security settings.

### Via Obtainium (automatic updates)

If you use **Obtainium** or a similar app manager, you can add Maikudoku to receive automatic updates directly from the repository.

## 🎲 How to Play

**Sudoku is super simple:**

1. You have a 9×9 grid divided into 9 blocks of 3×3, with some numbers already filled in.
2. Fill the empty cells with numbers from 1 to 9, making sure that:
   - Each row contains the numbers 1–9 (only once each).
   - Each column contains the numbers 1–9 (only once each).
   - Each 3×3 block contains the numbers 1–9 (only once each).
3. When you've correctly filled all the cells, you win! 🏆

Tap a cell to select it, then use the numbers at the bottom to enter your guess. If you make a mistake, the number turns red – no worries, just delete it and try again!
